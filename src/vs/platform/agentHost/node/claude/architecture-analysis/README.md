# Claude Agent Host — Architecture Analysis

Working analysis of the coupling, lifecycle, and duplication friction across the
four core files of the Claude agent host runtime:

- `claudeAgent.ts` — the `IAgent` provider (owns sessions, sequencers, chat backings)
- `claudeAgentSession.ts` — per-session/per-chat coordinator (materialize, diffs, registries)
- `claudeSdkPipeline.ts` — owns one SDK `Query` lifecycle (rebind, config replay)
- `claudePromptQueue.ts` — the prompt iterable + deferred bookkeeping
